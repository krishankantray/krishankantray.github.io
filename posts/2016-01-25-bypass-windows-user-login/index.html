Bypass Windows User Login

2016-01-25 :: Krishankant Ray

Bypass Microsoft Windows User Login    ⌗

Step I : Get any Linux Distribution live bootable disk (generally all Linux Distros comes         with live boot option).   

If you don’t have bootable disk download it from the respective website. Although Ubuntu is most popular Linux distribution. You can download Ubuntu from :http://www.ubuntu.com/download/desktop   ( select the latest version according to your cpu artitecture viz. 32 bit or 64 bit). After download, you can burn it DVD or a USB flash drive).

**
**

Step II: Now reboot your PC and enter into BIOS menu by pressing particular function key depending on your your pc manufacturer ( for lenovo its F12 and for HP its F10). 

In the boot menu select DVD (if you want to boot from Linux DVD) or select USB name (if you want to boot from usb drive.

Step III : Select live boot option ( in some cases like linux mint, it automatically executes live boot without giving any option). Once the desktop loads, first of all open COMPUTER icon available on desktop region. (see the desktop region of linux mint)




After opening my computer,  open the disk on which your windows is loaded, then select windows folder ( see fig. below)



   Inside windows folder you will find ‘system32’ folder, just open it and rename sethc.exe to anything.exe and rename cmd.exe to sethx.exe

Step III: Start your PC after removing bootable device and wait for windows login screen to appeat.

Step IV: When windows login screen appear, press shift 5 times, then command prompt appears on the screen. 

**Type the following : ( assuming that the username is Jack and password as 1234 )

net user Jack 1234**

hit enter and exit command prompt.

Then just login without typing any password. 

THATS DONE…………….

try it …..have good luck… **
